# BRAND BRIEF — Ecohome.net

## Logo Analysis
- **Format:** Inline SVG with two groups: `icon-container` (geometric arrow/recycling shape) and `name-container` (wordmark: "eco" in bold + "HOME" in lighter weight)
- **Logo Icon Color:** `#F05240` (coral red) — via CSS variable `--logo-icon-color`
- **Logo Text Color:** `#4E4F48` (dark olive/charcoal) — via CSS variable `--logo-text-color`
- **Logo Font:** Wordmark uses custom paths (not a font), style is modern sans-serif with "eco" bold and "HOME" lighter
- **SVG file saved:** `/public/logo.svg`
